\## What it is



A personal portfolio and blog site built with Hugo and hosted 

for free on GitHub Pages.



\## Why I built it



I wanted a place to document my learning, share projects, 

and have a professional presence online without paying for hosting.



\## Tech used



\- Hugo (static site generator)

\- PaperMod theme

\- GitHub Pages (free hosting)

\- GitHub Actions (auto-deploy)



\## What I learned



\- How static site generators work

\- Git basics — add, commit, push

\- How GitHub Actions automates deployment

\- Markdown for writing content
